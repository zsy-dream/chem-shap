## app/ — 应用主目录

### 核心文件

| 文件 | 说明 |
|------|------|
| `__init__.py` | Flask 应用工厂，注册蓝图与扩展 |
| `models.py` | 数据库模型定义（User / Sample / ModelRecord / AnalysisReport） |
| `cli.py` | Flask CLI 命令（数据库初始化等） |

### app/routes/ — 路由层

| 文件 | 说明 |
|------|------|
| `web.py` | **Web 前端路由**：首页仪表盘、样本管理页、模型页、分析页、报告页；定义 8 维实验特征标签 |
| `auth.py` | 用户认证：登录/注册/登出 |
| `data.py` | 数据接口：样本的增删改查、CSV批量导入、数据校验 |
| `model.py` | 模型接口：XGBoost/RF/LightGBM 训练、GridSearchCV调优、模型列表与删除 |
| `analysis.py` | 归因分析接口：SHAP归因计算、4种可视化（瀑布图/力导向图/摘要图/依赖图） |
| `report.py` | 报告接口：报告生成、查看、评级系统（优秀/良好/待优化） |
| `health.py` | 健康检查接口 |

### app/services/ — 业务逻辑层

| 文件 | 说明 |
|------|------|
| `data_service.py` | 数据服务：样本清洗、统计、CSV解析 |
| `model_service.py` | 模型服务：训练流程、超参搜索、模型持久化 |
| `attribution_service.py` | **核心服务**：SHAP归因计算（TreeExplainer/KernelExplainer）、4种图表生成、智能优化建议生成 |
| `evaluation_service.py` | 评估服务：8项指标计算（AUC/F1/Precision/Recall/Sensitivity/Specificity/PPV/NPV）+ Youden's J |
| `report_service.py` | 报告服务：综合报告生成、评级逻辑、趋势分析 |

| `Samples.html` | 实验样本列表页（分页、搜索、批量操作） |

| `models.html` | 模型管理页（训练、对比、删除） |

| `reports.html` | 报告列表页 |
| `report_detail.html` | 报告详情页（评级仪表盘、特征贡献排序、对比图） |

## 8 维实验特征字段

系统围绕以下 8 个化学实验条件进行归因分析与优化：

| 字段名 | 含义 | 单位 |
|--------|------|------|
| `reaction_temperature` | 反应温度 | ℃ |
| `reaction_time_min` | 反应时间 | min |
| `ph_value` | pH 值 | — |
| `catalyst_loading` | 催化剂负载量 | % |
| `solvent_polarity` | 溶剂极性指数 | 0-10 |
| `stirring_speed_rpm` | 搅拌速度 | rpm |
| `reactant_ratio` | 反应物配比 | — |
| `crystallization_time_min` | 结晶时间 | min |
